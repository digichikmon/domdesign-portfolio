What I offer

Services

Design work built on production knowledge — every deliverable is meant to function in the real world, not just on screen.

01

Branding & Visual Identity

Wordmarks, monograms, typographic systems, and the brand logic that ties it together. I start with the typographic structure and build outward — color, signage, stationery, and digital applications. Every identity system is designed to hold up across contexts and scales, from a business card to a building sign.

Logo design Wordmark Brand system Style guide Stationery Signage applications
02

Graphic Design & Print Production

Brochures, posters, flyers, event materials, and production-ready print files. Built from a print background — bleeds, color profiles, registration, and the discipline of files that run cleanly the first time through the press. I know what your printer needs before you have to ask.

Brochures & booklets Posters & banners Event materials Pre-press & file prep Offset & digital print
03

Technical Design & Evacuation Plans

Technical drafting, floor-plan layouts, evacuation route diagrams, and code-compliant safety signage. Produced to the same standards as my daily work at PMU Québec — legible under stress, in poor lighting, and across different print sizes. Every plan is a working document that has to function before it can look good.

Evacuation plans Floor plan layout Symbol systems AutoCAD drafting Safety signage Code compliance
04

Web Design & Digital Assets

Clean, structured HTML/CSS websites for small studios, professionals, and local businesses — designed with the same grid discipline I apply to print. Fast, accessible, and built to last without relying on platforms that change their pricing every year.

Portfolio sites Studio websites HTML & CSS GitHub Pages Digital assets Social media graphics
